# AI-Driven-Climate-Aware-Irrigation-Scheduling-System
Built an AI-driven irrigation prediction model using BiLSTM and BiRNN on NASA POWER climate data (2011–2025). Applied Bayesian hyperparameter tuning to optimize performance (lower RMSE) and generated 2026–2040 forecasts (RCP 4.5), capturing temporal patterns for intelligent irrigation scheduling.

Integrated ET₀ estimation (FAO-56 Penman–Monteith) and soil moisture balance modeling for accurate water demand prediction.
Compared deep learning architectures, with BiLSTM outperforming BiRNN in capturing long-term temporal dependencies.
Generated season-specific (Rabi & Kharif) irrigation schedules to support practical agricultural decision-making.
Designed a scalable, data-driven pipeline for climate-resilient irrigation planning.
